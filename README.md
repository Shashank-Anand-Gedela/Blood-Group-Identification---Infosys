## 📌 About Project

This project is a web-based application developed using Django. It helps in identifying a person's blood group based on given input data. The main goal of this project is to simplify the prediction process and provide quick results through a web interface.
This project was developed as part of my Infosys Springboard Internship, where I gained hands-on experience in full-stack development using Python and Django.

## 🚀 What this project does

This application performs the following tasks:

* Takes input data from the user through a web form
* Processes the input using backend logic
* Applies conditions or logic to determine blood group
* Displays the result instantly on the screen
* Ensures smooth interaction between frontend and backend

## 🛠️ Technologies Used

The following technologies were used to build this project:

* Python → Used for backend logic and processing
* Django → Used as the web framework to build the application
* HTML → Used to design the structure of web pages
* CSS → Used for styling and improving UI
* SQLite → Used as the database to store data

## ▶️ How to Run

Follow these steps to run the project:

1. Clone or download this repository
2. Open the project folder in terminal
3. Install required dependencies
4. Run the Django server
5. Open browser and access the application
